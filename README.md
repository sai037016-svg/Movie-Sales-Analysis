# Movie Sales Analysis – Power BI Dashboard Project:
# Project Overview:

This project focuses on analyzing movie sales data using Power BI to uncover revenue trends, customer purchasing behavior, and category-level performance. The dashboard transforms raw sales data into interactive visual insights, enabling stakeholders to monitor performance and make informed business decisions.

# Business Objective:

The entertainment industry relies heavily on data to understand audience preferences and maximize revenue. This project aims to:

Analyze movie sales performance across categories and time periods

Identify top-performing movies and revenue contributors

Track sales trends and patterns

Support data-driven business strategy and decision-making

# Approach & Methodology:

Imported and transformed raw sales data using Power Query

Cleaned and structured the dataset for analysis

Created calculated measures and KPIs using DAX

Designed interactive dashboards with charts, KPI cards, and filters

Implemented visualizations to highlight trends and performance metrics

# Skills Demonstrated:

Data modeling and transformation

DAX calculations and KPI creation

Interactive dashboard design

Data visualization and storytelling

Business insight generation

# Tools & Technologies Used:

Microsoft Power BI

Power Query

DAX (Data Analysis Expressions)

Data Visualization Techniques

# Key Insights:

Identified top-performing movies contributing the highest revenue

Observed sales trends across different time periods

Highlighted category-level performance differences

Provided insights into revenue distribution and sales patterns

This project demonstrates my ability to design professional dashboards, analyze sales data effectively, and present insights in a clear and actionable format for both technical and non-technical stakeholders.
